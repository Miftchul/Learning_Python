## How to Use

1. Clone the repository:
    ```bash
    git clone https://github.com/Miftchul/Learning_Python.git
    ```
2. Navigate to the `Belajar_15` folder:
    ```bash
    cd Learning_Python/Belajar_15
    ```
3. Run the Python scripts:
    ```bash
    python script_name.py
    ```
